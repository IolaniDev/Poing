# Contributing to iobot

Wanna help contribute to iobot development?  We'd love to see your code become a part of our project!  

Wanna contribute to help us code iobot?  We welcome pull requests for bug fixes and feature requests/implementations, provided they pass our continuous integration tests and don't negatively impact the core functionality of iobot.  Please open an issue for any bugs you find or features you would like to see implemented.
